# Titanic Chatbot

## Overview

A chatbot application designed to provide information and interactive discussions about the Titanic disaster.

## Features

- Interactive Q&A about Titanic history
- Passenger information lookup
- Historical facts and timeline
- Natural language processing for user queries

## Installation

```bash
git clone <repository-url>
cd titanic-chatbot
fastapi dev backend/main.py
# in another terminal
streamlit run frontend/app.py
```

## Usage

```bash
npm start
```

## Technologies

- Python
- Natural Language Processing
- FastAPI
- Streamlit
